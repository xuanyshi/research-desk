关于我

我是 Xuanyu Shi，方法学研究者，工作在统计学、数据科学与流行病学的交叉地带——本科统计学、硕士数据科学（均在美国），博士就读于北京大学医学部（健康医疗大数据方向）。

研究上主要关注：因果推断方法（Pearl 的 DAG/SCM 传统，以及 Hernán–Robins 的流行病学设计，如 target trial emulation）、生物医学知识图谱、跨研究设计的证据三角互证（RCT / 孟德尔随机化 / 观察性研究）、真实世界证据；应用侧多在药物流行病学与肿瘤流行病学（如 GLP-1RA 与肿瘤、消化系统肿瘤的危险因素）。

代表性成果：Nature Communications（2025，一作，Evidence Triangulator）、Scientific Data（2024，一作）、Health Data Science（2024）。

关于这个站点

这个站点就干两件事。

一是每天替我盯着一批高水平期刊。它把新文章抓下来，让模型照我的兴趣打个分、写段中文摘要，攒成「每日速览」。分高的我才细看。

二是存我自己挑出来的东西。值得读的收进「文献库」，标上标签、打个分、记下读到哪、写几句笔记。真读透了的，我会另写一篇精读。

说明一下：「每日速览」里的分数和摘要是大模型生成的；文献库的笔记和每篇精读，都是我自己写的。

怎么用

抓最近的新文章并打分：

python -m pipeline.daily

收藏一篇：

python -m pipeline.add 2401.12345 --tags 因果推断 --rating 4 --status read --note "第3节重点"

改了数据或模板，只重新生成页面：

python -m pipeline.render

搭在什么上面

页面是 Python 加 Jinja2 直接生成的静态 HTML，放在 GitHub Pages 上。追新除了盯白名单期刊（Crossref），也用 OpenAlex 按主题搜全网已发表文献、并追 arXiv/medRxiv 预印本；打分走一个 OpenAI 兼容的模型接口。每天的抓取、每次的部署，都交给 GitHub Actions。

源码和配置说明都在 README 里。
